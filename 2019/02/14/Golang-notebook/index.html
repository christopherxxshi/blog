变量定义

  1. var 变量名 变量类型 //可以默认为空

  1. 定义的变量必须使用

  2. 可以自动推断类型

  1. 第一次定义可以使用:= (不可以在函数外使用)

内建变量类型

  1. bool, string
  2. (u)int (长度根据系统字节), (u)int8, (u)int16,(u)int32, (u)int64, uintptr
  3. byte, rune(go中的char, 32位)

  4. float32, float64, complex64(复数，实部32位，虚部32位), complex128

  5. 类型转换是强制的, 没有隐式转换

常量与枚举

  1. 常量定义 const 常量名 （类型） = 值

  1. 常量不要随意大写
  2. 常量数值可作为任意数值类型使用

  3. 枚举

条件语句

  1. if 条件不需要括号
  2. if 条件里可以赋值， 赋值的变量作用域就在这个if语句里

  3. switch 可以自动break, switch 后边可以没有变量， 在case中写表达式

循环

  1. for 的条件里不需要括号， 可以省略初始条件，结束条件， 递增表达式
