变量类型与计算

题目

  1. typeof 能够得到哪些类型
  2. === 与 ==

知识点

值类型 vs 引用类型 (按存储方式类型区分)

  • 值类型： 只进行值传递
  • 引用类型： 指针传递
  • object, function, array 都是引用类型
  • 引用类型可以无限扩展属性

typeof 运算符

  1. typeof 只能区分出值类型的具体类型
    • typeof undefined // undefined
    • typeof ‘abc’ // string
    • typeof 123 // number
    • typeof true // boolean

  2. typeof引用类型中能区分出来function

    • typeof {} // object
    • typeof [] // object
    • typeof null // object
    • typeof console.log // function

      强制类型转换

  3. 字符串拼接

    • 100 + ’10’ // ‘10010’
  4. == 运算符
    • 100 == ‘100’ // true 100 -> ‘100’
    • 0 == ‘’ // true 0-> false ‘’ -> false
    • null == undefined // true null -> false undefined -> false
  5. if 语句
    • b = 100
    • if (b) // true
  6. 逻辑运算
    • 10 && 0 // 0
    • “” || “abc” // “abc”

=== vs ==

  1. jQuery 中推荐的写法
    • obj.a == null
    • 相当于 obj.a == null || obj.a == undefined
  2. 其他地方全用 ===
